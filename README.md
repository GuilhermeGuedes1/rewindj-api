# Orbit

AI-Powered Event Management Platform

Orbit is a SaaS platform designed for event agencies and artists.

## Features

- Multi-tenant architecture
- JWT Authentication
- Role-based access control
- Event management
- Client management
- AI-powered event parsing using Gemini
- Swagger API documentation
- PostgreSQL + Prisma
- Docker deployment

## Tech Stack

- NestJS
- TypeScript
- PostgreSQL
- Prisma
- Docker
- Gemini AI
